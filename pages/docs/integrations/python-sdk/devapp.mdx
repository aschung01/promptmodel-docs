# DevApp

Promptmodel **DevApp** connects your codebase to the development dashboard.

## Methods

### register

A decorator to register PromptModel usage inside a function to the development dashboard.

```python
@client.register
def my_function():
    ...
    outputs = PromptModel("my_prompt").run_and_parse(inputs=INPUTS)
    # my_prompt will be displayed in the development dashboard
    ...
```

### register_function

| Parameter   |                   Type                   | Required | Description                                                                                       |
| :---------- | :--------------------------------------: | -------: | ------------------------------------------------------------------------------------------------- |
| description | Optional[Dict[str, Any], FunctionSchema] |      Yes | Description of the function to be passed on as the schema required for function calls.            |
| function    |                 Callable                 |      Yes | Function to be registered and used for test runs (function calls) from the development dashboard. |

### register_sample

| Parameter |      Type      | Required | Description                                          |
| :-------- | :------------: | -------: | ---------------------------------------------------- |
| name      |      str       |      Yes | Name of the sample to be displayed in the dashboard. |
| content   | Dict[str, Any] |      Yes | Dictonary of an input sample.                        |

### include_client

| Parameter |  Type  | Required | Description                                    |
| :-------- | :----: | -------: | ---------------------------------------------- |
| client    | Client |      Yes | A client instance to include in the dashboard. |
