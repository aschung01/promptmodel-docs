# Introduction

**PromptModel** is a prompt engineering framework that offers (or will offer) the following:

- [x] Prompt editor (with diff visualization) synced with your code
- [x] Cloud management for promptmodels in production
- [ ] A/B test management _(coming soon)_
- [ ] Metric visualization for development and production _(coming soon)_

We are currently in **closed alpha**. You can try us out [here](https://app.promptmodel.run).

You can request a personal demo [here](https://calendly.com/sounhochung/30min).

Stay updated by following us on [Discord](https://discord.gg/dVTtyEU4).
